# BasicRAG-Project
A basic test to construct a local RAG for an LLM.   
This project is based on Track B of the assignment.   

# Basic Usage
Install required dependencies:    
  
    pip install -r requirements.txt    
    
Gather an initial list of 1000 python methods and store them within a RAG.    
    
    Run python run_part1.py    

Continue using the RAG set up previously and add additional methods, stored within data\custom_functions, to test what impact this will have on the RAG model.

    Run python run_part2.py
